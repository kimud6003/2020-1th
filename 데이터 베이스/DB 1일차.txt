1장 데이터베이스 시스템의 세계

Data : 자료/정보 컴퓨터에 의해서 표혀되어지는 자료들
	
	-structured data: 테이블에 저장되어있는 데이터
	-unstructured data: 나열되어있는 데이터

Database: 조직화된 데이터의 묶음

DBMS: DB의 관리하는 시스템 소프트웨어 

데이터베이스 시스템
	데이터베이스 + DBMS



DBMS의 간단한기능
	-영구적인 저장
	-편리한 프로그래밍 인터페이스
	-트랜잭션 관리(동시 접근 지원)

DBMS의 기능
	-데이터베이스와 스키마(데이터 베이스의 구조)의 생성
	-query를 사용및 검색 변경
	-대용량 데이터의 관리
	-다중 사용자의 지원
	-작은 크기의 많은 양의 데이터 처리에 효과적이다


데이터 모델의 변화
	파일 -> 계층 모델,네트워크 모델
	관계형 모델 ()
